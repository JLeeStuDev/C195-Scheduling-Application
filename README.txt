Title: STUdev Scheduling Application

Purpose:
The STUdev Scheduling application is a JavaFX-based application desinged to manage appointments and customer information for a fictional software development company.  It allows users to schedule and manage appointments, track customer information, and log user activity.

Author:				Jonathan Michael Lee
Contact Information:		jlee921@wgu.edu
Student Application Version: 	1.0
Date:				March 23rd, 2024

IDE: IntelliJ IDEA Community Edition 2023.2.2
JDK: Java SE 17.0.1
JavaFX Version: JavaFX-SDK-17.0.1

Directions for Running the Program:
1. Download the zip folder containing the application files.
2. Extract the contents of the zip folder to your desired location on your computer.
3. Open IntelliJ IDEA.
4. From the IntelliJ IDEA menu, select "File" > "Open..." and navigate to the directory where you extracted the zip folder.
5. Select the project directory and click "Open" to load the project.
6. Configure the project SDK to Java SE 17.0.1 and set up the JavaFX SDK.
7. Build and run the Main class to start the application.

Additional Report Description:
For the additional report, I created a billing report, which provides a summary of the payments made by customers.
